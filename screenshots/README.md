# Screenshots Guide

To complete the README documentation, please add the following screenshots to the `screenshots/` directory:

## Required Screenshots

1. **landing-page.png** - Homepage showing hero section, vehicle types, and how it works
2. **rider-step1.png** - Rider booking form Step 1 (When?)
3. **rider-step2.png** - Rider booking form Step 2 (Where?)
4. **rider-step3.png** - Rider booking form Step 3 (Vehicle selection)
5. **rider-step4.png** - Rider booking form Step 4 (Confirm)
6. **booking-status.png** - Booking tracking page
7. **driver-registration.png** - Driver registration form
8. **driver-dashboard.png** - Driver dashboard with assigned rides
9. **admin-dashboard.png** - Admin panel showing bookings
10. **analytics.png** - Analytics dashboard

## How to Capture Screenshots

1. Run the development server: `npm run dev`
2. Navigate to each page
3. Take screenshots (use browser dev tools for consistent sizing)
4. Save them in the `screenshots/` directory with the exact names listed above
5. Recommended size: 1200px width for consistency

## Pages to Screenshot

- `/` - Landing page
- `/rider` - Rider booking (capture all 4 steps)
